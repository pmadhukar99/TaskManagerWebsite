## 🚀 Complete Free Setup (15 Minutes)

### Step 1: Create Firebase Project (5 mins)

1. Go to [firebase.google.com](https://firebase.google.com)
2. Click **Get Started**
3. Click **Create a project**
4. Name: `TaskManager`
5. Click **Create Project**
6. Wait for project to create

### Step 2: Set Up Firestore Database (3 mins)

1. In Firebase, click **Firestore Database**
2. Click **Create database**
3. Choose **Start in test mode** (free)
4. Select location closest to you
5. Click **Create**

### Step 3: Get Firebase Config (2 mins)

1. Click **Settings** (gear icon)
2. Click **Project settings**
3. Scroll down to **Your apps**
4. Click **</>** (Web app)
5. Copy the Firebase config
6. Save for later
